대표님, 요청하신 전해질 제재의 투여 효과와 바실러스 서브틸리스(Bacillus subtilis)의 항바이러스 효과에 대한 시험 데이터를 정리해 드립니다.  
먼저, **전해질 제재 및 바실러스 서브틸리스(DNG-1000 제품군 기준)의 효능 시험 데이터**를 바탕으로 구성한 표입니다.

### **1\. 주요 성분별 효능 시험 결과 요약**

| 분류 | 시험 항목 | 시험 조건/내용 | 결과 및 효능 |
| :---- | :---- | :---- | :---- |
| **항바이러스** | 바실러스 서브틸리스 (DNG-1000) | 바이러스 외피 당단백질 복제 억제 | DNJ 성분 생성 및 바이러스 감염능력 사멸 |
| **전해질/성분** | 전해질 및 기호성 제재 (파보겔) | 설사 환축 대상 경구 투여 | 탈수 방지 및 빠른 회복, 높은 기호성 |
| **효능 비교** | BVDV 억제 효과 | 정제 DNJ vs 바실러스 MORI 배양물 | MORI 배양물이 정제 DNJ보다 30배 적은 양으로도 우수한 효과 |
| **열안정성** | 온도별 생균수 유지율 | 80℃(5초), 100℃(60초) 처리 | 100℃ 고온에서도 높은 생균수 유지 |

### **2\. 특정 바실러스 서브틸리스(Bacillus subtilis MORI) 항바이러스 효과**

DNG-1000의 핵심 균주인 *Bacillus subtilis MORI*가 생산하는 DNJ(1-deoxynojirimycin)의 바이러스 계통별 억제 효과입니다.

| 바이러스 계통 | 대상 바이러스 | 효과 |
| :---- | :---- | :---- |
| **Arteriviruses** | PRRSV (돼지생식기호흡기증후군) | 효과 확인 |
| **Orthomyxoviridae** | 조류독감 A (AIV) | 효과 확인 |
| **Pestiviruses** | 소바이러스성설사병 (BVDV) | 효과 확인 |
| **Bunyaviridae** | 아까바네, 아이노 바이러스 | 효과 확인 |
| **Rhabdoviridae** | 소유행열 바이러스 (BEF) | 효과 확인 |

(참고: 상기 데이터는 한국아그로 DNG-1000 관련 문서 내 시험 결과를 인용하였습니다.)
